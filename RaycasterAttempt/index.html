I gave the settings no maximum or minimum. Experiment, but remember it might break if you go overboard, if it does just refresh the page

FOV | degrees of view
FPS | frames per second/gameSpeed
render accuracy | how many lines per frame you get, set to lower values for better performance
turn sensitivity | how many degrees you turn per frame
step length | how many blocks you walk per frame
render distance | how many blocks you can see forward
game speed | how fast the game is, default 1000 lower is faster
sprint rate | how many times faster is sprinting to walking
noclip | walking through walls
pause framecount | pause animations ect.
